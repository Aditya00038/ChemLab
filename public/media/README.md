# Media Assets

## App Logo

Place your `app_logo.png` file in this directory.

**Requirements:**
- File name: `app_logo.png`
- Format: PNG with transparent background recommended
- Recommended size: 512x512px or larger (square aspect ratio)
- Will be displayed at various sizes (16x16, 32x32, 180x180, 192x192, 512x512)

The logo will appear in:
- **Browser Tab/Favicon** (16x16, 32x32)
- **Dashboard header** (24x24 - top left)
- **Mobile menu header** (24x24)
- **PWA App Icon** (192x192, 512x512)
- **Apple Touch Icon** (180x180)

**Note**: After adding or updating the logo:
1. Restart the development server
2. Clear browser cache or hard refresh (Ctrl+Shift+R)
3. The logo will appear in the browser tab next to "ChemStock"

If no logo is present, the browser will show a default icon.
